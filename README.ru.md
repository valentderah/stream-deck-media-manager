# Current Media

<img alt="image" src="ru.valentderah.current-media.sdPlugin/imgs/banner.png" />

Плагин для Stream Deck, который отображает информацию о текущем воспроизводимом медиа и предоставляет элементы управления воспроизведением.

## Ручная установка

1. Скачайте последнюю версию со страницы [Releases](https://github.com/valentderah/media-manager/releases)
2. Распакуйте папку `ru.valentderah.current-media.sdPlugin`
3. Откройте приложение Stream Deck
4. Перейдите в магазин Stream Deck (меню из трёх точек) и нажмите "Больше действий" → "Установить с диска"
5. Выберите папку `ru.valentderah.current-media.sdPlugin`

## Возможности

- Плагин отображает информацию о текущем медиа с обложкой альбома, названием и исполнителями. (Есть режим работы в 2x2 клетки)
- Дает возможность перехода между медиа и установка/снятие с паузы.

## Требования

- **Elgato Stream Deck Software**: 6.9 или выше
- **Операционная система**: Windows 10 или выше
- **Медиаплееры**: Любое приложение, совместимое с Windows Media Session API (включая Spotify, Yandex Music, VLC, Windows Media Player и многие другие)
